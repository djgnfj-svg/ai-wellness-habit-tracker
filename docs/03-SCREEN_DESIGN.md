## 🎯 **사용자 여정 (User Journey)**

### **첫 방문 사용자**
```
1. 앱 다운로드 → 2. 온보딩 → 3. 카카오 로그인 → 4. 프로필 설정 → 5. 습관 추천 → 6. 첫 체크인
   (30초)        (2분)      (30초)        (3분)        (2분)       (1분)
```

### **일반 사용자 (매일)**  
```
1. 앱 실행 → 2. 홈 대시보드 → 3. 습관 체크인 → 4. AI 코칭 확인 → 5. 완료
   (즉시)      (10초)         (30초)        (1분)          (즉시)
```

### **주간 사용자**
```
1. 홈 → 2. 통계 확인 → 3. 습관 추가/수정 → 4. AI 코칭 대화 → 5. 설정 변경
```
